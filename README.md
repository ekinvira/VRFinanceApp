# VR Finance App 💸

Modern bir gelir - gider ve envanter yönetim uygulaması.

## 🚀 Özellikler
- Gelir / Gider takibi
- Dashboard (Toplam gelir, gider, net kazanç)
- Son eklenen kayıtlar
- Raporlar (aylık filtreleme)
- Envanter yönetimi

## 🛠️ Teknolojiler
- WPF (.NET 8)
- Entity Framework Core
- SQLite

## 🎯 Amaç
Farklı sektörlerde kullanılabilecek genel bir finans takip uygulaması geliştirmek.
